## خطوات التشغيل

### 1. تحديث قاعدة البيانات
```bash
# تشغيل الترحيل
npx prisma db push

# أو إنشاء ترحيل جديد
npx prisma migrate dev --name add-purchase-system
```

### 2. إضافة البيانات التجريبية
```bash
# تشغيل سكريبت البيانات التجريبية
psql $DATABASE_URL -f src/lib/seed-purchase-system.sql
```

### 3. الوصول للنظام
- صفحة الشراء: `/purchase`
- API الخدمات: `/api/purchase/services`
- API الحزم: `/api/purchase/packages`

## هيكل قاعدة البيانات

### الجداول الجديدة
1. **ConsultantAvailability** - جدول توفر المستشارين
2. **Purchase** - جدول المشتريات
3. **Reservation** - جدول الحجوزات

### التحديثات على الجداول الموجودة
- **Consultant**: إضافة معدل الساعة والسيرة الذاتية
- **Service**: إضافة السعر ومدة الخدمة
- **User**: إضافة رقم الهاتف
- **subscription_packages**: إضافة نسبة الخصم
