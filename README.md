# ConsultPro - Business Consulting Platform

A professional consulting website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **SEO Optimized**: Comprehensive meta tags, semantic HTML, and structured data
- **Professional Design**: Modern gradient-based UI with smooth animations
- **Responsive**: Mobile-first design that works on all devices
- **Accessible**: ARIA labels, screen reader support, and keyboard navigation
- **Static**: No backend dependencies - purely client-side

## Pages

- **Home (/)**: Hero section, services overview, stats, and CTA
- **Services (/services)**: Detailed consulting service offerings
- **Blog (/blog)**: Business insights and expertise articles
- **Contact (/contact)**: Professional contact form and information

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React 18

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
